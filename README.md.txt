RECIPE SEARCH ON THE DJANGO WEBSITE
 This is django food recipe application. 
 This application is built with python’s best web application framework django  and Css Framework Bootstrap 
 In this application, Admin will add recipes with image, title and description.
 Frontend user will see the all recipes and also find their favroite recipe with search functionality.
Before running the django project, please make sure that you have python 3 installed in your system beucase django is python framework and run with python.

Python and Django are important for this project 

Steps to run this project

1. To begin with, install Python and Django.

2 installed packages that are required.

3 pip install -requirements.text file.

4 use Python manage.py runserver
